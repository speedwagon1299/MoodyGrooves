# MoodyGrooves

Clone the repo
```
git clone https://github.com/speedwagon1299/MoodyGrooves.git
cd MoodyGrooves
```

To run backend (Port: 4000)
```
cd backend
npm ci
npm run build
npm start
```

To run frontend (Port 4173)
```
cd frontend
npm ci
npm run build
npm run preview
```
